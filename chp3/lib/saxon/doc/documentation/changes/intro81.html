Version 8.1

This page lists changes in Saxon 8.1 (released 2004-09-24) relative to version 8.0. The list applies both to the open source Saxon-B 8.1 product and to Saxon-SA, the schema-aware product.

As far as possible this release reflects the W3C specifications up to and including the July 2004 working drafts.

Note in particular the change in namespace URIs: URIs that started with http://www.w3.org/2003/11/ now start http://www.w3.org/2004/07/

Some of the changes made in this release reflect Working Group decisions made subsequent to the publication of the July 2004 drafts. In most cases these decisions have been announced publicly on the public-qt-comments list.

Saxon 8.1 is tested with JDOM 1.0 rc 1. It also continues to work with JDOM beta 10.

For changes made in version 8.0, see Version 8.0.

For a change history prior to version 8.0, see the SourceForge site.

The changes in version 8.1 are listed under the following headings:

  • Differences between Saxon-SA and Saxon-B

  • Defects Cleared

  • XSLT changes

  • Changes to XQuery

  • Changes to the XQuery Java API

  • Changes to Schema-Aware XSLT and XQuery Processing

  • Changes to Schema Processing

  • Changes to Functions and Operators

  • Changes to Extension Functions

  • Internal Changes

  • Changes to Extension Function Calling Mechanism

  • Changes to Serialization
