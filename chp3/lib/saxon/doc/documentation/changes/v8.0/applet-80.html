Running Saxon as an Applet

The XSLTProcessorApplet module, and the sample applet applications, are withdrawn. It was difficult to configure Saxon to run as an applet, and as far as I can see, no-one is using the facility. The JAR file has become uncomfortably large for use in an applet.

It remains possible to invoke Saxon as an applet, but you will have to create your own interface to do so.
