Version 8.1.1

Version 8.1.1 is a maintenance release: its main purpose is to clear a couple of significant bugs that were found to be present in 8.1.

In addition, the Saxon-SA version of the product also includes a redesigned front-end for the schema processor, designed to improve the way checking of XML representation constraints is done.

The changes in version 8.1.1 are listed under the following headings:

  • Defects cleared

  • Changes in XML Schema support

  • JAXP Application Programming Interface

  • Changes in Functions and Operators
